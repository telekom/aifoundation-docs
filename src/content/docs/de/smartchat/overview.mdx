T-Systems SmartChat ist eine KI-gestützte Konversationsschnittstelle mit integrierter Retrieval-Augmented Generation (RAG). Es ist in die LLM-Serving-Infrastruktur integriert und liefert präzise, kontextbezogene Antworten auf Basis Ihrer Dokumente.


## Hauptfunktionen

- **End-to-End-Hosting** — Verwalten Sie Ihre Unternehmensdokumentenbasis mit umfassendem Hosting
- **RAG-Zugriff** — Laden Sie Dokumente hoch und fragen Sie diese per UI oder API ab
- **Anpassbar** — Über 50 Retrieval-Einstellungen zur Optimierung für Ihren Anwendungsfall
- **24/7-Support** — Rund-um-die-Uhr-Betriebsunterstützung
- **Planbare Preise** — Monatliche Pauschalgebühren

## Erste Schritte

Um SmartChat zu nutzen, kontaktieren Sie das AIFS-Team unter **[ai@t-systems.com](mailto:ai@t-systems.com)** für Zugangsdaten. Nach der Einrichtung können Sie:

1. Sich über die API authentifizieren
2. Dokumente hochladen und verarbeiten
3. Chat-Sitzungen erstellen
4. Ihre Dokumente mit KI abfragen

Eine vollständige Anleitung finden Sie im [Local Chat](/de/smartchat/local-chat)-Leitfaden.

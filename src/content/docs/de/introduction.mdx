AI Foundation Services ist eine private, sichere und skalierbare Plattform für Modellinferenz. Unsere Mission ist es, Ihnen Tools bereitzustellen, um mit den neuesten KI-Innovationen Schritt zu halten. Ob Sie ein Startup sind, das KI in seine Produkte integriert, oder ein Unternehmen mit bestehenden Anwendungen – wir möchten Sie unterstützen.

Entdecken Sie unsere Kernangebote:
- [**LLM Hub Serving API:**](#llm-hub-serving-api) Zugang zu einer Vielzahl von Open-Source- und proprietären LLMs über eine OpenAI-kompatible API.
- [**Fine Tuning API:**](#fine-tuning-api) Passen Sie LLMs mit eigenen Daten an, um optimale Leistung für Ihren spezifischen Anwendungsfall zu erzielen.
- [**T-Systems SmartChat:**](#t-systems-smartchat--smartchat-api) Erweitern Sie Ihre Konversationen mit einer intuitiven Benutzeroberfläche, die Dokumenten-Uploads und RAG unterstützt.
- [**SmartChat API:**](#t-systems-smartchat--smartchat-api) Integrieren Sie leistungsstarke KI-Funktionen, einschließlich RAG-Funktionalität, in Ihre Anwendungen.

Entdecken Sie unsere Tools zur Verbesserung Ihrer Erfahrung:
- [**LLM Playground:**](#llm-playground) Optimieren und experimentieren Sie mit verschiedenen Modellen in einer benutzerfreundlichen Umgebung.
- [**API-Key-Portal:**](#api-key-portal) Verwalten Sie einfach Ihre API-Keys und überwachen Sie Nutzung und Kosten.
- [**Create-TSI:**](#create-tsi) Vereinfachen Sie die Implementierung von KI-Modellen mit unserem umfassenden Toolset.

## Kernangebote

### LLM Hub Serving API
Unsere LLM Hub Serving API bietet flexible Lösungen für Ihre KI-Anforderungen. Ihre Daten werden in Deutschland und der EU gehostet, vollständig konform mit der DSGVO. Wählen Sie das Setup, das Ihren Anforderungen entspricht:

**Shared Instance:**
- **Standardisierte APIs:** Zugang zu den neuesten LLMs über OpenAI-kompatible APIs.
- **Gemeinsame Ressourcen:** Nutzen Sie gemeinsame Ressourcen für effiziente Modellinferenz.
- **Kosteneffiziente Preisgestaltung:** Profitieren Sie von einem nutzungsbasierten Preismodell.

**Private Instance:**
- **Exklusives Modell-Hosting:** Hosten Sie private Modelle auf der Open Telekom Cloud.
- **Dedizierte Ressourcen:** Stellen Sie optimale Leistung mit Ihren Modellen gewidmeten Ressourcen sicher.
- **Fine-Tuning-Service:** Verbessern Sie Ihre Modelle mit unserem optionalen Fine-Tuning-Service.
- **Monatliche Gebühren:** Profitieren Sie von planbaren Gebühren für eine konsistente Budgetierung.

**Cloud-Optionen:**  
Wir bieten die Bereitstellung auf mehreren Cloud-Plattformen für optimale Leistung und Flexibilität:
- **Open Telekom Cloud:** Zugang zu Open-Source-LLMs, die sicher in Deutschland oder der Schweiz gehostet werden.
- **MS Azure/AWS/GCP:** Verbindung zu proprietären Modellen führender Cloud-Anbieter.

### Fine Tuning API
Unsere Fine-Tuning-API ermöglicht es, Modelle mit eigenen Daten an Ihre Bedürfnisse anzupassen. Erzielen Sie hohe Anpassung und Genauigkeit mit minimalem Aufwand.

**Hauptmerkmale:**
- **Erweitertes Fine-Tuning:** Nutzen Sie Techniken wie Low-Rank Adaptation (LoRA) oder Direct Policy Optimization (DPO) mit Reinforcement Learning from Human Feedback (RLHF).
- **Einfacher Daten-Upload:** Laden Sie Ihre Trainingsdaten hoch, um Modelle anzupassen.
- **Nahtlose Integration:** Integration in bestehende Systeme durch OpenAI-API-Kompatibilität.
- **Flexible Bereitstellung:** Stellen Sie trainierte Modelle auf Shared oder Private Instances bereit.

---

### T-Systems SmartChat & SmartChat API
Unsere T-Systems SmartChat-Benutzeroberfläche ist ein Overlay, das sich in die LLM-Serving-Infrastruktur integriert und die Nutzung von KI-Modellen vereinfacht. Laden Sie Dokumente hoch und nutzen Sie Retrieval-Augmented Generation (RAG) für präzise Interaktionen.

Für erweiterte Integrationen nutzen Sie die SmartChat API mit vollem Zugang zum RAG-System. Erstellen Sie dynamische Konversationsschnittstellen für den Kundenservice oder interne Wissensdatenbanken.

**Hauptmerkmale:**
- **End-to-End-Hosting:** Verwalten Sie Ihre Unternehmensdokumentenbasis mit umfassendem Hosting.
- **RAG-Zugang:** Nutzen Sie Funktionen wie Dokumenten-Upload und RAG über UI oder API.
- **Anpassung und Bewertung:** Passen Sie über 50 Abrufeinstellungen an und bewerten Sie diese.
- **24/7-Support:** Verlassen Sie sich auf Rund-um-die-Uhr-Support für reibungslosen Betrieb.
- **Monatliche Gebühren:** Profitieren Sie von planbaren Gebühren für eine konsistente Budgetierung.

AI Foundation Services stellt eine OpenAI-kompatible API bereit. Das bedeutet, dass die meisten Frameworks und Werkzeuge, die für OpenAI entwickelt wurden, sofort einsatzbereit sind — passen Sie einfach die Basis-URL auf `https://llmhub.t-systems.net/v2` an und geben Sie Ihren API-Schlüssel an. Die folgenden Seiten beschreiben die gängigsten Framework-Integrationen mit funktionsfähigen Einrichtungsbeispielen.

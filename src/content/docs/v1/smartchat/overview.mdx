# SmartChat RAG

T-Systems SmartChat is an AI-powered conversational interface with built-in Retrieval-Augmented Generation (RAG). It integrates with the LLM Serving infrastructure to provide accurate, context-aware responses using your documents.

---

## Key Features

- **End-to-End Hosting** — Manage your company document base with comprehensive hosting
- **RAG Access** — Upload documents and query them via UI or API
- **Customizable** — Over 50 retrieval settings to tune for your use case
- **24/7 Support** — Round-the-clock operational support
- **Predictable Pricing** — Monthly flat-rate fees

## Getting Started

To use SmartChat, contact the AIFS team for access credentials. Once set up, you can:

1. Authenticate via the API
2. Upload and ingest documents
3. Create chat sessions
4. Query your documents with AI

See the [Local KB Guide](/smartchat/local-chat) for a complete walkthrough.
